# Prepare

Persipaan sebelum run

## Require
PHP 8.2

MySql

## Installation
Setelah Clone lakukan update composer

```bash
composer update
```

Jika tidak bisa 
```bash
composer install
```
kemudian jika sudah lakukan migrasi database

```bash
php artisan migrate
```
